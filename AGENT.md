# Анализ рынка, ориентированного на сбыт

## Цель

Выполнить задание для студентов магистратуры экономического факультета: провести количественный анализ рынка, ориентированного на сбыт.

## Структура задания

1. **Формулирование проблемы и дизайн исследования** — эксплоративное исследование (вторичное), анализ фактических данных из внутренних и внешних источников. Формулирование гипотез.
2. **Выбор источников информации и методов сбора данных**
3. **Операционализация и измерение переменных**
4. **Сбор данных**
5. **Анализ данных:**
   - Однофакторный анализ (дескриптивный: табулирование, графики, статистические параметры; заключающий: z-тест, t-тест, хи2-тест)
   - Многофакторный анализ (факторный, кластерный, неметрическое многомерное шкалирование)
6. **Оценка данных и интерпретация результатов**
7. **Составление отчёта и презентация результатов**

## Структура отчёта

1. Формулировка целей исследования
2. Описание данных (количество наблюдений, переменных, тип данных, характер распределения)
3. Обоснование и применение методов исследования
4. Описание полученных результатов и их интерпретация

### Пример данных (sources/Анализ+данных+ПащенкоТВ+Банкроты+ПК.xlsx)

Пример аналогичного исследования на данных компаний-банкротов Пермского края. Содержит:

- **Лист "report"** — полная финансовая отчётность ~250 компаний за 2020–2024 гг.: баланс (активы, пассивы), отчёт о финансовых результатах (выручка, себестоимость, коммерческие/управленческие расходы, прочие доходы/расходы, чистая прибыль), налоги
- **Лист "Лист2"** — сводная таблица по видам деятельности (133 отрасли)
- **Лист "Выборка 1"** — выборка ключевых финансовых показателей: основные средства, запасы, дебиторская задолженность, денежные средства, нераспределённая прибыль, заёмные средства, кредиторская задолженность, выручка, себестоимость, коммерческие и управленческие расходы, прочие доходы/расходы, чистая прибыль
- **Лист "Обработка"** — расчёт средних абсолютных изменений ключевых показателей (14 переменных)
- **Листы "Описательные статистики"** — дескриптивная статистика переменных
- **Листы "Корреляция", "Корреляция 2"** — корреляционный анализ
- **Листы "Регрессия", "Регрессия 2"** — регрессионный анализ
- **Листы "Данные для регрессии", "Данные для регрессии (2)"** — подготовленные данные

### Методические примеры (sources/Примеры для отчета (1).pptx)

Рекомендации по написанию отчёта:

- **Визуализация** — гистограммы, ящик с усами, диаграммы рассеяния
- **Дескриптивная статистика** — средняя, медиана, стандартное отклонение, асимметрия, эксцесс
- **Выбор метода по типу переменной**: номинальная → таблицы частот, хи-квадрат; порядковая → ранговая корреляция; интервальная → корреляция Пирсона, регрессия
- **Условия применимости**: корреляция Пирсона (линейность, без выбросов), t-критерий (нормальность, равенство дисперсий), МНК (условия Гаусса-Маркова)
- **Пример**: анализ зависимости прибыли от оборота магазинов (нелинейная зависимость, важность графиков)
- **Принцип**: выводы должны быть конкретными, на профессиональном языке, с обоснованием
