# Этап 2. Источники информации и методы сбора данных

## Источник данных

**ГИРБО (Государственный информационный ресурс бухгалтерской (финансовой) отчётности)** — bo.nalog.gov.ru.

Ресурс ведётся ФНС России и содержит бухгалтерскую отчётность всех юридических лиц, обязанных её представлять. Данные публичны и доступны через API.

### Обоснование выбора

- **Официальный источник** — отчётность представляется в ФНС в обязательном порядке, данные верифицированы
- **Единый формат** — стандартизированные формы бухгалтерского баланса и отчёта о финансовых результатах
- **Полнота охвата** — доступна отчётность за 5 лет (2021–2025), включая малые предприятия
- **Программный доступ** — API позволяет автоматизировать сбор и обеспечить воспроизводимость

### Метод сбора

Автоматизированный сбор через REST API ГИРБО (скрипт `collect_data.py`):

1. Поиск организации по ИНН: `GET /advanced-search/organizations/search?query={ИНН}`
2. Загрузка отчётности всех периодов: `GET /nbo/organizations/{id}/bfo/`
3. Извлечение данных баланса (`balance`) и отчёта о финансовых результатах (`financialResult`)
4. Консолидация в единый датасет

## Период наблюдений

**2021–2025 гг.** (5 лет).

Выбор обусловлен:
- 2021 — первый полный год после снятия ковидных ограничений, начало восстановления рынка массовых мероприятий
- 2025 — последний доступный отчётный период
- 5 лет — достаточный горизонт для оценки динамики и трендов

## Собранные показатели

### Бухгалтерский баланс (17 показателей)
- Нематериальные активы, основные средства, внеоборотные активы
- Запасы, дебиторская задолженность, краткосрочные финансовые вложения, денежные средства, оборотные активы
- Активы всего
- Уставный капитал, нераспределённая прибыль, капитал и резервы
- Заёмные средства (долгосрочные и краткосрочные), кредиторская задолженность
- Долгосрочные и краткосрочные обязательства

### Отчёт о финансовых результатах (13 показателей)
- Выручка, себестоимость продаж, валовая прибыль
- Коммерческие и управленческие расходы, прибыль от продаж
- Доходы от участия в других организациях, проценты к получению/уплате
- Прочие доходы и расходы
- Прибыль до налогообложения, чистая прибыль (убыток)

## Ограничения метода

1. **Только вторичные данные** — анализ ограничен публичной бухгалтерской отчётностью, не включает управленческие данные (количество мероприятий, число участников, средний чек)
2. **Юрлицо ≠ бренд** — одна компания может работать через несколько юрлиц, одно юрлицо может вести несколько видов деятельности
3. **Разные формы отчётности** — ООО сдают стандартную форму, АНО и ассоциации — форму НКО (КНД 0710096), что ограничивает сопоставимость отдельных показателей
4. **Упрощённая отчётность** — часть малых предприятий сдаёт сокращённую форму без детализации отдельных статей
