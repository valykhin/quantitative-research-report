# Этап 6. Многофакторный анализ

## 6.1. Регрессионный анализ

**Модель:** Валовая рентабельность, % = a + b · ln(Выручка)

Логарифм выручки выбран как предиктор для линеаризации связи (абсолютные значения выручки имеют правую асимметрию).

| Параметр | Значение |
|---|---|
| N | 36 |
| Intercept (a) | −4.18 |
| Slope (b) | +1.40 |
| R² | 0.031 |
| p-value (slope) | 0.301 |

**Модель незначима** (p = 0.30, R² = 0.03) — логарифм выручки объясняет лишь 3% дисперсии рентабельности.

### Проверка условий МНК

| Условие | Результат |
|---|---|
| Нормальность остатков (Шапиро-Уилк) | W = 0.989, p = 0.971 — ✓ выполнено |
| Среднее остатков ≈ 0 | 0.000 — ✓ выполнено |
| Независимость остатков от предиктора | r = 0.000, p = 1.000 — ✓ выполнено |

Условия Гаусса-Маркова выполнены — модель корректна, но предиктор не объясняет вариацию зависимой переменной.

### Визуализация

![regression](plots/regression.png)

**Вывод:** линейная зависимость рентабельности от масштаба бизнеса отсутствует. Рентабельность определяется иными факторами — бизнес-моделью, эффективностью управления, структурой мероприятий.

## 6.2. Кластерный анализ (H4)

### Переменные кластеризации

| Переменная | Обоснование |
|---|---|
| Выручка | Масштаб бизнеса |
| Валовая рентабельность, % | Эффективность основной деятельности |
| Оборачиваемость активов | Интенсивность использования ресурсов |
| Доля КЗ в активах, % | Зависимость от заёмного финансирования |
| Доля ДС в активах, % | Ликвидность |

Данные стандартизированы (z-score). Использован последний доступный период по каждой компании. N = 8 компаний.

### Выбор числа кластеров

| k | Инерция | Силуэт |
|---|---|---|
| 2 | 27.02 | 0.170 |
| **3** | **18.19** | **0.211** |
| 4 | 12.63 | 0.166 |
| 5 | 7.90 | 0.131 |

Оптимальное k = 3 (максимальный силуэт), но силуэт = 0.211 < 0.3 — **кластерная структура выражена слабо**.

### Состав кластеров

| Кластер | Компании | Профиль |
|---|---|---|
| 0 | ПАРКРАН, Петербургский спорт, Арена Плюс, АСМ «Новый Спорт» | Малые/средние, низкая рентабельность (~5%), низкая доля КЗ (~24%), низкая ликвидность |
| 1 | Архитектура спорта (IRONSTAR), ГРУТ | Средние, средняя рентабельность (~11%), высокая доля КЗ (~66%), **высокая ликвидность (~78%)** |
| 2 | Лига героев, Марафон Сервис | Крупные, высокая рентабельность (~16%), высокая доля КЗ (~67%), низкая ликвидность |

### Визуализация

![clusters](plots/clusters.png)

### Интерпретация

Несмотря на низкий силуэт, кластеры содержательно интерпретируемы:

- **Кластер 0 — «Малые организаторы»**: компании на ранних стадиях или с нишевой моделью. Низкая рентабельность, низкая зависимость от поставщиков.
- **Кластер 1 — «Кэш-генераторы»**: средние по выручке, но с высоким запасом денежных средств. Модель с предоплатами от участников.
- **Кластер 2 — «Лидеры рынка»**: крупнейшие по выручке, наиболее рентабельны, но с высокой кредиторской нагрузкой.

### Вердикт по H4

**H4 не подтверждена формально** (силуэт 0.211 < 0.3), но кластеры содержательно различимы. Слабая кластерная структура объясняется малым размером выборки (8 компаний) — при расширении выборки кластеры могут стать статистически устойчивыми.
