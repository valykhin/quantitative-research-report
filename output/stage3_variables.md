## Типы переменных и выбор методов

| Тип переменной | Переменные | Дескриптивный анализ | Тесты связи |
|---|---|---|---|
| Номинальная | Компания, ОПФ | Таблица частот, круговая диаграмма | Хи-квадрат, таблицы сопряжённости |

## Обоснование выбора методов анализа

### Дескриптивный анализ

| Метод | Почему выбран | Применение |
|---|---|---|
| Описательные статистики (среднее, медиана, σ, min, max) | Базовая характеристика распределения; медиана устойчива к выбросам при малой выборке | Все интервальные переменные |
| Асимметрия и эксцесс | Определяют, какую меру центральной тенденции использовать: при сильной асимметрии — медиану, при симметричном распределении — среднее | Все интервальные переменные |
| Гистограммы | Визуальная оценка формы распределения и выявление мультимодальности | Выручка, рентабельность, активы |
| Boxplot (ящик с усами) | Наглядное выявление выбросов и сравнение распределений между компаниями/периодами | Все ключевые показатели |
| Линейные графики динамики | Визуализация трендов во времени — критично для проверки H1 (устойчивый рост) | Выручка, активы, прибыль по годам |

### Инференциальный анализ

| Метод | Почему выбран | К какой гипотезе |
|---|---|---|
| Тест Шапиро-Уилка | Проверка нормальности — определяет, применимы ли параметрические тесты; подходит для малых выборок (n < 50) | Предварительный шаг |
| t-тест (одновыборочный) | Проверка отличия среднего от нуля; применяется если подтверждена нормальность | H1: средний темп роста > 0 |
| Тест Вилкоксона (знаковых рангов) | Непараметрическая альтернатива t-тесту при нарушении нормальности; не требует допущений о распределении | H1: если нормальность не подтверждена |
| Корреляция Пирсона | Оценка линейной связи между интервальными переменными; требует нормальности и линейности | H2, H3, H5 |
| Корреляция Спирмена | Ранговая альтернатива Пирсону; устойчива к выбросам и нелинейным монотонным связям; предпочтительна при малой выборке | H2, H3, H5: если нормальность нарушена |
| Хи-квадрат | Проверка связи между номинальными переменными | ОПФ vs прибыльность (при дихотомизации) |

### Многофакторный анализ

| Метод | Почему выбран | К какой гипотезе |
|---|---|---|
| Линейная регрессия (МНК) | Моделирование зависимости результативного показателя от предикторов; оценка направления и силы влияния | H3: доля себестоимости от выручки |
| Проверка условий МНК | Гаусс-Марков: нормальность остатков (Шапиро-Уилк), гомоскедастичность (визуально), отсутствие автокорреляции; без проверки выводы ненадёжны | Валидация регрессии |
| Кластерный анализ (k-means) | Выявление групп компаний с похожим финансовым профилем; интерпретируемость результатов | H4 |
| Метод локтя + силуэтный коэффициент | Обоснование числа кластеров вместо произвольного выбора | H4: оптимальное k |
| Стандартизация (z-score) | Переменные имеют разные масштабы (от % до млн руб.); без стандартизации кластеризация будет определяться переменной с наибольшим разбросом | Подготовка к кластерному анализу |

### Логика выбора метода в зависимости от данных

```
Переменная интервальная?
  ├─ Да → Проверка нормальности (Шапиро-Уилк)
  │        ├─ Нормальное → Пирсон, t-тест, МНК-регрессия
  │        └─ Ненормальное → Спирмен, Вилкоксон, осторожность с регрессией
  └─ Нет (номинальная) → Таблицы сопряжённости, хи-квадрат
```
