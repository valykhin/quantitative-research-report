# Сбор и анализ финансовой отчётности

Проект для сбора финансовой отчётности компаний-организаторов беговых мероприятий из открытого API ГИРБО (bo.nalog.gov.ru) и последующего анализа.

## Требования

- Python 3.12+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Быстрый старт

```bash
# Установить зависимости
uv sync

# 1. Собрать данные из ГИРБО
uv run python collect_data.py

# 2. Подготовить аналитический датасет (производные показатели)
uv run python prepare_dataset.py

# 3. Дескриптивный анализ (статистики, тесты, графики)
uv run python descriptive_analysis.py

# 4. Инференциальный анализ (проверка гипотез, корреляции)
uv run python inferential_analysis.py

# 5. Многофакторный анализ (регрессия, кластеры)
uv run python multivariate_analysis.py
```

### Результаты

|---|---|---|
| `collect_data.py` | `data/companies_financials.xlsx`, `data/raw_bfo.json` | Сырая финансовая отчётность 11 компаний за 2021–2025 |

## Скрипты

### collect_data.py

1. Загружает список компаний из `companies.csv` (для добавления новой компании — добавить строку `name,inn` в этот файл)
2. По каждому ИНН ищет организацию через API `bo.nalog.gov.ru/advanced-search/organizations/search`
3. Для каждой найденной организации запрашивает бухгалтерскую отчётность (БФО) через `bo.nalog.gov.ru/nbo/organizations/{id}/bfo/`
4. Извлекает из ответа данные баланса и отчёта о финансовых результатах за все доступные периоды
5. Сохраняет консолидированную таблицу в `data/companies_financials.xlsx` и сырые данные в `data/raw_bfo.json`

### Расширение выборки

Чтобы добавить новую компанию, достаточно дописать строку в `companies.csv`:

```csv
Название бренда,ИНН
```

Затем перезапустить весь пайплайн — `prepare_dataset.py` и `descriptive_analysis.py` подхватят новые данные автоматически.

### prepare_dataset.py

Готовит аналитический датасет на основе сырых данных:
1. Рассчитывает производные показатели: рентабельность (валовая, по ЧП), доли в активах (ОС, ДС, ДЗ, КЗ), коэффициент автономии, оборачиваемость активов
2. Вычисляет темпы роста год к году (выручка, активы, КЗ)
3. Добавляет индикатор ОПФ (ООО / НКО)
4. Рассчитывает средние абсолютные изменения по компаниям

### descriptive_analysis.py

Выполняет однофакторный (дескриптивный) анализ:
1. Таблицы частот (ОПФ, покрытие данных)
2. Описательные статистики (среднее, медиана, σ, асимметрия, эксцесс, IQR)
3. Проверка нормальности (тест Шапиро-Уилка)
4. Графики: гистограммы, boxplot по компаниям, динамика показателей, темпы роста, сводка рынка

### inferential_analysis.py

Выполняет инференциальный (заключающий) анализ — проверку гипотез и анализ взаимосвязей:
1. H1: тест Вилкоксона на положительность медианного темпа роста выручки
2. H2, H3, H5: корреляции Спирмена (масштаб vs рентабельность, эффект масштаба, рост выручки vs рост КЗ)
3. Корреляционная матрица по 9 ключевым показателям с p-values
4. Хи-квадрат тест (ОПФ vs прибыльность)
5. Графики: тепловая карта корреляций, scatter-графики для гипотез

### multivariate_analysis.py

Выполняет многофакторный анализ:
1. Линейная регрессия (валовая рентабельность от ln(выручки)) с проверкой условий МНК
2. Кластерный анализ (k-means) по 5 финансовым показателям с выбором оптимального k (метод силуэта)
3. PCA-визуализация кластеров и профили кластеров

## Структура выходного файла

Каждая строка — одна компания за один год. Столбцы:

- **Компания**, **ИНН**, **Период** — идентификация
- **Баланс**: основные средства, запасы, дебиторская задолженность, денежные средства, активы всего, капитал и резервы, заёмные средства, кредиторская задолженность и др.
- **Отчёт о финансовых результатах**: выручка, себестоимость, прочие доходы/расходы, чистая прибыль (убыток) и др.
