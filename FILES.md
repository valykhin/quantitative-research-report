# Структура проекта и описание файлов

## Управление проектом

| Файл | Описание |
|---|---|
| `AGENT.md` | Цель проекта, исходные данные, план реализации, ссылки на все артефакты |
| `IMPLEMENTATION_PLAN.md` | Поэтапный план (9 этапов) с чеклистами шагов и статусами выполнения |
| `REQUIREMENTS.md` | 39 требований к приёмке работы с отметками выполнения (38/39 выполнено) |
| `FILES.md` | Этот файл — описание всех файлов проекта |

## Исходные материалы (`sources/`)

| Файл | Описание |
|---|---|
| `data.txt` | Список компаний для анализа (бренд + ИНН) |
| `Задание на исследование рынка, ориентированного на сбыт (1).docx` | Методическое задание |
| `Примеры для отчета (1).pptx` | Рекомендации по написанию отчёта, примеры визуализации и анализа |

| `СПАРК_Выборка_компаний_*.xlsx` | Дополнительные выборки из СПАРК |

## Python-проект (`app/`)

### Конфигурация

| Файл | Описание |
|---|---|
| `README.md` | Инструкция по запуску: установка, пайплайн, описание скриптов |
| `pyproject.toml` | Зависимости Python-проекта (pandas, scipy, sklearn, matplotlib и др.) |
| `companies.csv` | Список компаний для сбора данных (name, inn) — единственный файл для редактирования при расширении выборки |
| `.python-version` | Версия Python (3.12+) |

### Скрипты

| Файл | Что делает | Команда |
|---|---|---|
| `collect_data.py` | Сбор финансовой отчётности из API ГИРБО по списку из `companies.csv` | `uv run python collect_data.py` |
| `prepare_dataset.py` | Расчёт производных показателей (рентабельность, темпы роста, доли и др.) | `uv run python prepare_dataset.py` |
| `descriptive_analysis.py` | Описательные статистики, тест нормальности, графики | `uv run python descriptive_analysis.py` |
| `inferential_analysis.py` | Проверка гипотез (Вилкоксон, Спирмен, хи-квадрат), корреляционная матрица | `uv run python inferential_analysis.py` |
| `multivariate_analysis.py` | Регрессия (МНК) и кластерный анализ (k-means) | `uv run python multivariate_analysis.py` |
| `build_report.py` | Сборка итогового xlsx-отчёта (10 листов с данными, расчётами и графиками) | `uv run python build_report.py` |

### Данные (`app/data/`)

| Файл | Описание |
|---|---|
| `companies_financials.xlsx` | Сырая финансовая отчётность 11 компаний за 2021–2025 (49 строк, 33 столбца) |
| `raw_bfo.json` | Сырые данные из API ГИРБО (JSON) |
| `analysis_dataset.xlsx` | Полный аналитический датасет с производными показателями (49 строк, 46 столбцов) |
| `analysis_revenue_subset.xlsx` | Выборка с данными по выручке (38 строк, 9 компаний) |
| `mean_changes.xlsx` | Средние абсолютные изменения показателей по компаниям |

### Результаты расчётов (`app/output/`)

| Файл | Описание |
|---|---|
| `report.xlsx` | **Итоговый отчёт** — 10 листов с данными, таблицами, графиками (самодостаточный для презентации) |
| `descriptive_stats.xlsx` | Описательные статистики (среднее, медиана, σ, асимметрия, эксцесс, IQR) |
| `normality_tests.xlsx` | Результаты теста Шапиро-Уилка |
| `hypothesis_tests.xlsx` | Сводка тестов гипотез H1–H5 |
| `correlation_matrix.xlsx` | Корреляционная матрица Спирмена + p-values |
| `regression_results.xlsx` | Коэффициенты регрессии, R², проверка условий МНК |
| `cluster_results.xlsx` | Принадлежность к кластерам, средние по кластерам |
